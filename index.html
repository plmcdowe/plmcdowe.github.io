  1. Work

    • Cisco and Bash: Dynamic on-the-box configuration of IOS XE routers and switches

      • Router-Config 'RTRCFG.sh': Configure a STIG baselined IOS XE C8300 in ~10.3 seconds with error handling.

      • Switch-Config 'SWCFG.sh': Configure a STIG baselined IOS XE C9200 in ~60 seconds with error handling.

      • Switch-Fix 'SWFIX.sh': Reconfigure an IOS XE C9200 deployed on the production network to STIG baseline.

      • DHCP-Pool 'CAPWAP.sh': Configures more than 60 DHCP pools on a C8000 in less than 5 seconds.

    • PowerShell and Forescout: WIN 11 security & compliance Powershell switch-case scripts executed by Forescout policy.

      • Application switch-case 'APP_SwitchCase.ps1': Performs .EXE; and installer; and Package searches using regex cases, then removes those items.

      • PNP switch-case 'PNP_SwitchCase.ps1': 2 functions based on hostname with 3 nested switches covering cases for -

        PNP status, Vendor ID, and Product ID, disabling prohibited drivers.

    • Cisco and Python: Dynamic reconfiguration of IOS XE routers and switches with the Netmiko library.

      • Guard Root 'GuardRoot.py': Correctly reconfigure spanning-tree guard root on switches deployed in production.

      • Router sub-int ACL's 'SubINT-ACL.py': Modify extended ACL's on a router's sub-int's to 1st deny traffic destined to itself & last - 'deny any any log-input'

      • Debug VPM 'debug_vpm_all.py': Automate identification of in-service POT's lines and reconfigure dial-peers and FXO interfaces on IOS XE routers.

      • Nested dictionary of site IP's 'ip_nestedDict.py': Dictionary of sites where the value's are lists of lists containing network device IP's.

    • Cisco and Event Manager: Introduction to Event Manager Applets for IOS XE router and switch configuration.

    • Cisco Syslog with Grafana: Config files and documentation to deploy open-source Grafana, Loki, and Promtail.

    • Cisco Nexus 9K: Config template for data center N9K switches.

    • Cisco MDS 9000 Fibre Channel (under construction): Config template for SAN fibre channel switches.
